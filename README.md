# 🚀 Mahmoud Hussein | Full-Stack Developer Portfolio

A premium, high-performance developer portfolio designed with a **"Glass Terminal"** aesthetic. Built for speed, accessibility, and visual impact.

![Portfolio Preview](public/og-image.png)

## ✨ key Features

- **💎 Glass Terminal Design**: Clean, modern UI using advanced CSS glassmorphism and backdrop blurs.
- **⚡ Pro Performance (90+ Score)**:
  - Code Splitting via `React.lazy` & `Suspense`.
  - Image optimization with native `lazy-loading`.
  - Hardware-accelerated animations using `will-change`.
  - Optimized mouse tracking with Framer Motion `useMotionValue` (zero re-renders).
- **📱 Responsive Navigation**: Seamless multi-page routing with `react-router-dom` and a custom mobile menu.
- **🎨 Rich Animations**:
  - **GSAP**: Orchestrated entrance timelines.
  - **Framer Motion**: Smooth 3D card tilts, floating tech icons, and hover effects.
- **🔍 SEO & Meta Optimized**:
  - Full Open Graph (OG) & Twitter Card support.
  - JSON-LD Structured Data for search engines.
  - Automated `sitemap.xml` and `robots.txt`.
- **✉️ Direct Communication**:
  - Integrated "Contact Me" form with `mailto:` link.
  - Floating WhatsApp action button with pulsing effect.
- **🖱️ Custom Experience**:
  - Glowing Custom Cursor.
  - Scroll Progress Bar.
  - Animated "Back to Top" button.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/), [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router v7](https://reactrouter.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/) (FontAwesome 6, Simple Icons)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mahmoudhussein6/My_Portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
